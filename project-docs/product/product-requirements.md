# Product Requirements

## League Creation
Users can create a league by entering a league name.

System generates:
- Game PIN
- Join link

## Join League
Players join using:
- Game PIN
- Nickname

## Draft
Each player drafts:
- 1 Hero
- 1 Villain
- 2 Cinderella teams

## War Room
Displays:
- League standings
- Player portfolios
- Score updates

## Bracket
Interactive bracket showing tournament results.

## Admin Controls
Admin can:
- Start draft
- Force start league
- Enter game results
- Lock league at tournament start
