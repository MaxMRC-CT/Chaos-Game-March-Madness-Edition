# AI Project Template

Use this repo as a master starter template before beginning any new AI-assisted software project.

## Folder Structure

- `ai-context/` — core project context for agents
- `product/` — product strategy and UX definitions
- `tasks/` — current sprint, backlog, bugs
- `prompts/` — reusable system and dev prompts
- `decisions/` — architectural and product decisions log
- `agents/` — agent roles, rules, and handoff guidance
- `docs/` — system maps and supporting documentation
- `notes/` — scratch notes and temporary thinking

## Recommended Setup Flow

1. Duplicate this template for a new project.
2. Rename the root folder to your project name.
3. Fill out the core files first:
   - `ai-context/project-overview.md`
   - `product/product-requirements.md`
   - `ai-context/architecture.md`
   - `ai-context/tech-stack.md`
   - `ai-context/coding-standards.md`
   - `ai-context/constraints.md`
4. Then fill out the 10x files:
   - `docs/system-map.md`
   - `docs/memory.md`
   - `decisions/decision-log.md`
   - `agents/agents.md`
   - `tasks/current-sprint.md`
5. Load the repo into your AI coding tool and tell the agent to read the markdown files before making changes.
