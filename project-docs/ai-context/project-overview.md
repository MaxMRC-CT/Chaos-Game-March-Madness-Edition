# Project Overview

## Project Name
Chaos League

## Purpose
Chaos League is a multiplayer March Madness strategy game where players draft a portfolio of teams rather than filling out a traditional bracket.

Players draft:
- Hero
- Villain
- Cinderella teams

These picks earn or lose points as the NCAA tournament progresses.

## Target Users
Friends playing in private leagues during March Madness.

## Core Experience
Users join leagues using a Game PIN, draft their teams, and track live standings throughout the tournament.

## Key Features
- Private leagues
- Draft-based gameplay
- Live War Room scoreboard
- Tournament bracket viewer
- Mobile-first experience

## Deployment
The application is deployed on Vercel and currently in beta testing.

## Design Goals
- Extremely easy to join and draft
- Exciting, competitive experience
- Mobile-first UI
- Fast load times
